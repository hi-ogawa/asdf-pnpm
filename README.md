# asdf-pnpm

```sh
# install plugin
asdf plugin add pnpm https://github.com/hi-ogawa/asdf-pnpm

# development
rm -rf ~/.asdf/plugins/pnpm
ln -srf ./ ~/.asdf/plugins/pnpm
asdf install pnpm latest
```
